# ✈️ TravelBloom — Discover the World

> A beautifully designed travel discovery webpage with real-time destination search, animated hero section, and 38+ curated destinations across the globe.

![TravelBloom Hero](https://images.unsplash.com/photo-1506197603052-3cc9c3a201bd?w=1200&q=85&auto=format&fit=crop)

---

## 🌟 Features

- **Real-time Search** — Instantly filter 38+ destinations by name, country, tags, or descriptor
- **Search Autocomplete** — Dropdown recommendation list with emoji icons and tags
- **Clear Button** — One-click × button to reset the search input
- **Country Browse Mode** — Type "countries" to browse all destinations grouped by country
- **Full Results Overlay** — Hit Enter or click Search to open a full-screen results panel
- **Ken Burns Hero** — Cinematic background animation with a glassmorphism search bar
- **Sticky Navbar** — Blurred, translucent navbar that reacts to scroll
- **Featured Destinations Grid** — 6 handpicked cards with scroll-in animations
- **About & Contact Sections** — Animated stat cards and a functional contact form
- **Responsive Design** — Mobile-friendly with a hamburger menu
- **Dark Mode First** — Deep dark palette with vibrant purple accents

## 🔍 Search Guide

| Query | Result |
|---|---|
| `paris` | Matches destinations with "paris" in name/tags |
| `india` | Shows all Indian destinations |
| `temple` | Lists temple destinations worldwide |
| `beach` | Filters beach destinations |
| `countries` | Browse all destinations grouped by country |
